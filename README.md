# pigeon-ui

A customizable, themeable authentication widget for modern React applications.

## Installation

```bash
npm install pigeon-ui
```

## Peer Dependencies

This package requires the following peer dependencies:

- React
- React-dom

```bash
npm install react react-dom
```

## Packages

### [@pigeon-ui/auth-widget](./auth-widget/packages/authwidget/)

A fully customizable authentication widget component for React applications. AuthWidget provides ready-to-use login and signup forms with built-in theme support, style overrides, and authentication status feedback.

**Key features:**

- Three built-in themes (modern, light, dark)
- Granular style customization via props
- TypeScript support with full type definitions
- Built-in status feedback system with auto-timeout
- Lightweight with tree-shaking support
